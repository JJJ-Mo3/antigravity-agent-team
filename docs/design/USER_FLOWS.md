# User Flows

## 1. Create and Organize
1.  **User opens app.** Empty board loads (or persisted data).
2.  **User clicks "Add another list".** Enters inputs "To Do".
3.  **User clicks "Add another list".** Enters inputs "Doing".
4.  **User clicks "Add a card"** in "To Do".
    *   Input opens.
    *   User types "Task 1".
    *   Hits Enter.
    *   Card appears instantly. Input stays open for "Task 2".
5.  **User drags "Task 1"** to "Doing".
    *   Immediate visual update.
    *   Dropped in new position.

## 2. Edit Card Details
1.  **User clicks "Task 1".**
2.  Modal opens.
3.  **User clicks Description.** Textarea becomes editable.
4.  User types details and clicks Save.
5.  **User clicks a colored label square.** Label is added to card.
6.  User clicks X or outside modal to close.

# User Personas

## 1. The Solo Developer
*   **Bio:** A freelance developer or hobbyist building small projects.
*   **Goals:** Track feature requests, bugs, and todo items for a specific project.
*   **Pain Points:** Jira is too complex; text files are too messy.
*   **Needs:** Visual prioritization, easy drag-and-drop, persistent state without server setup.

## 2. The Personal Planner
*   **Bio:** Someone planning a trip, a wedding, or a home renovation.
*   **Goals:** organize myriad small tasks into categories (e.g., "To Buy", "To Do", "Done").
*   **Pain Points:** Sticky notes get lost; spreadsheet are not visual enough.
*   **Needs:** Simple interface, fast entry of items, clear visual progress.
